## 📋 v0.5.1 任务清单

### 1. 真实嵌入集成 - 替换 mock_embedding

**目标**: 接入真实嵌入 API，让语义搜索真正工作

**任务**:
- [ ] 实现 `OpenAIEmbeddingClient`（调用 text-embedding-3-small）
- [ ] 实现本地模型支持（BGE/M3E，可选）
- [ ] 替换 `src/core/context.rs` 中的 `mock_embedding()`
- [ ] 添加嵌入缓存（避免重复调用）
- [ ] 单元测试（10 个）

**预计工作量**: 3 天

---

### 2. Router 架构 - 实现多层路由

**目标**: 实现多层路由架构，支持 Agent 层级和权限管理

**任务**:
- [ ] 创建 `src/router/` 模块
- [ ] 实现 `Router` 结构体（id, level, parent, children, capabilities）
- [ ] 实现 `RouterManager`（生命周期管理）
- [ ] 实现 `RouterConnector`（路由间通信）
- [ ] 实现 `RouterLevel`（Top/Upper/Lower/Special）
- [ ] 单元测试（15 个）

**预计工作量**: 5 天

---

### 3. 安全增强 - SSRF 防护、提示注入检测

**目标**: 增强安全层，达到生产级防护

**任务**:

#### 3.1 SSRF 防护
- [ ] 创建 `src/security/ssrf.rs`
- [ ] 实现内网 IP 检测
- [ ] 实现域名白名单
- [ ] 集成到 `web_fetch` 工具
- [ ] 单元测试（5 个）

#### 3.2 提示注入检测
- [ ] 创建 `src/security/injection.rs`
- [ ] 实现恶意模式检测（正则匹配）
- [ ] 检测常见攻击模式：
  - "Ignore previous instructions"
  - "Translate to JSON"
  - "Act as if you are..."
- [ ] 集成到消息处理流程
- [ ] 单元测试（5 个）

**预计工作量**: 3 天

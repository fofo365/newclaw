## 📋 v0.5.2 任务清单

### 1. 语义搜索完善 - RAG 实现

**目标**: 实现完整的 RAG (Retrieval-Augmented Generation) 能力

**任务**:
- [ ] 实现 `RetrievalConfig`（top_k, threshold, rerank）
- [ ] 实现 `retrieve()` 方法（语义检索）
- [ ] 实现上下文构建（检索结果 + 原始查询）
- [ ] 添加引用和来源追踪
- [ ] 混合检索（关键词 + 语义）
- [ ] 单元测试（10 个）

**预计工作量**: 4 天

---

### 2. MCP 支持 - 工具服务器化

**目标**: 支持 MCP (Model Context Protocol) 工具服务器

**任务**:
- [ ] 创建 `src/mcp/` 模块
- [ ] 实现 `McpClient`（连接 MCP 服务器）
- [ ] 实现 `McpServer`（暴露工具为 MCP 服务）
- [ ] 工具发现和注册
- [ ] JSON-RPC 通信
- [ ] 单元测试（10 个）

**预计工作量**: 5 天

---

### 3. 执行沙箱 - 基础隔离

**目标**: 实现基础代码执行隔离

**任务**:
- [ ] 创建 `src/sandbox/` 模块
- [ ] 实现 `Sandbox` trait
- [ ] 实现基础进程隔离（Linux namespace）
- [ ] 资源限制（CPU、内存、时间）
- [ ] 安全审计
- [ ] 单元测试（8 个）

**可选（后续版本）**:
- [ ] WASM 沙箱支持
- [ ] Docker 容器隔离

**预计工作量**: 4 天

---

### 4. Skill 支持

**目标**: 兼容 OpenClaw Skill 系统

**任务**:
- [ ] 创建 `src/skill/` 模块
- [ ] 实现 `SkillLoader`（加载 SKILL.md）
- [ ] 实现 `SkillExecutor`（执行技能脚本）
- [ ] 技能发现和注册
- [ ] 技能权限管理
- [ ] 单元测试（8 个）

**预计工作量**: 3 天

### 5. 上下文压缩

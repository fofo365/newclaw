# NewClaw v0.7.0 - 任务流原生架构

**版本**: v0.7.0  
**代号**: TaskFlow  
**核心目标**: 任务流原生架构，实现序列化、断点续传、跨设备迁移

---

## 🎯 核心特性

### 1. 任务流原生架构
- **Task**: 核心抽象，支持序列化、断点续传、跨设备迁移
- **Session**: Task 的视图，管理任务链和上下文
- **Focus Chain**: 任务焦点链，管理任务优先级和执行顺序

### 2. DAG 工作流引擎
- **DagBuilder**: DAG 构建器，支持依赖声明和循环检测
- **DagExecutor**: DAG 执行器，拓扑排序执行
- **DagCheckpoint**: 断点续传支持

### 3. 6 层配置架构
- **Layer 1**: User（用户配置）
- **Layer 2**: Workspace（工作空间配置）
- **Layer 3**: Global（全局默认配置）
- **Layer 4**: Naming（命名匹配配置）
- **Layer 5**: Agent（Agent 级配置）
- **Layer 6**: Session（会话级配置）

### 4. 约束保护系统
- **Constraint**: 约束定义（硬约束/软约束/偏好）
- **ConstraintManager**: 约束管理器
- **ConstraintInjector**: 约束注入器

### 5. Layer 0 瞬时层
- **EphemeralContext**: 当前 LLM 调用上下文
- **TokenBudget**: Token 预算控制
- **AdaptiveAllocator**: 自适应分配器

### 6. 统一记忆存储
- **MemoryStorage**: 统一存储抽象
- **FTS5 全文索引**: BM25 + 向量混合检索
- **MMR 去重**: 最大边际相关性去重

### 7. 分层摘要机制
- **SummaryTree**: 分层摘要树
- **HierarchicalSummaryManager**: 长对话压缩

### 8. 向量存储
- **VectorStore Trait**: 向量存储抽象
- **InMemoryVectorStore**: 内存实现
- **SQLiteVectorStore**: SQLite 实现

### 9. 代码库嵌入索引
- **CodeIndex**: 代码库语义搜索
- **CodeChunk**: 代码分块
- **ChunkType**: 分块类型（函数/类/模块/文件）

## 🚀 使用示例

### 创建任务

```rust
use newclaw::task::{Task, TaskId, TaskScope, TaskPriority};

let mut task = Task::atomic("分析代码库".to_string())
    .with_description("分析项目结构和依赖关系".to_string())
    .with_scope(TaskScope::Project)
    .with_priority(TaskPriority::High);

task.start()?;
// ... 执行任务 ...
task.complete()?;
```

### 创建 DAG 工作流

```rust
use newclaw::task::{DagBuilder, DagExecutor, DagExecutorConfig};

let mut dag = DagBuilder::new("deployment-workflow")
    .add_node("test", "运行测试", "shell")
    .add_node("build", "构建项目", "shell")
    .add_node("deploy", "部署服务", "shell")
    .add_edge("test", "build")?
    .add_edge("build", "deploy")?
    .build()?;

let executor = DagExecutor::new(DagExecutorConfig::default());
let result = executor.execute(&mut dag, |id, task_type, params| {
    // 执行任务
}).await?;
```

### 使用约束系统

```rust
use newclaw::memory::{Constraint, ConstraintType, ConstraintScope};

let constraint = Constraint::hard(
    "不要删除用户数据".to_string(),
    ConstraintScope::Global,
);

manager.add_constraint(constraint).await?;
```

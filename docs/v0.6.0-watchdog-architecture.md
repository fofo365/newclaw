# NewClaw v0.6.0 - 分离式双主控架构

## 📋 版本概述

**版本**: v0.6.0  
**代号**: Watchdog  
**目标**: 实现高可用、自愈的 AI Agent 基础设施

## 🎯 核心特性

### 分离式双主控架构
- **核心主控（CoreController）**: 轻量级看门狗，独立进程，< 32MB 内存
- **智慧主控（SmartController）**: 业务逻辑处理，动态资源，50-200MB

### 三级恢复策略
- **L1 - 快速修复**: 重启服务、清理缓存、回滚配置（< 1s）
- **L2 - AI 诊断**: 调用 LLM 分析日志，生成修复脚本（5-30s）
- **L3 - 人工介入**: 告警通知，进入安全模式（立即）

### 关键机制
- **租约机制（Lease）**: 防止脑裂，15s 过期，10s 续约
- **心跳检测**: 5s 间隔，3 次失败触发恢复
- **指数退避**: 1s → 2s → 4s → 8s，避免恢复风暴
- **审计日志**: 完整决策链，JSON 格式

## 🔧 核心模块设计

### 1. 核心主控（CoreController）

**文件**: `src/watchdog/controller.rs`

```rust
pub struct CoreController {
    /// 租约管理器
    lease_manager: Arc<LeaseManager>,
    /// 心跳检测器
    heartbeat_checker: HeartbeatChecker,
    /// 诊断引擎
    diagnostic_engine: DiagnosticEngine,
    /// 恢复执行器
    recovery_executor: RecoveryExecutor,
    /// 审计日志
    audit_log: AuditLogger,
    /// 配置
    config: WatchdogConfig,
}

impl CoreController {
    /// 主循环 - 5s 间隔检测
    pub async fn run(&mut self) -> Result<()> {
        loop {
            // 1. 检查租约有效性
            if !self.lease_manager.is_valid() {
                self.enter_safe_mode("Lease expired").await?;
                continue;
            }
            
            // 2. 检测智慧主控心跳
            match self.heartbeat_checker.check().await {
                Ok(status) => {
                    if !status.is_healthy() {
                        self.trigger_recovery(status).await?;
                    }
                }
                Err(e) => {
                    self.handle_connection_failure(e).await?;
                }
            }
            
            tokio::time::sleep(self.config.check_interval).await;
        }
    }
    
    /// 触发分级恢复
    async fn trigger_recovery(&mut self, status: HealthStatus) -> Result<()> {
        // 记录审计日志
        self.audit_log.log_event(AuditEvent {
            event_type: EventType::RecoveryTriggered,
            component: "smart_controller".to_string(),
            details: format!("Status: {:?}", status),
        }).await;
        
        // 分析故障，生成恢复计划
        let recovery_plan = self.diagnostic_engine.analyze(&status).await?;
        
        // 执行恢复
        self.recovery_executor.execute(recovery_plan).await?;
        
        Ok(())
    }
}
```

### 2. 租约管理器（LeaseManager）

**文件**: `src/watchdog/lease.rs`

```rust
pub struct LeaseManager {
    /// 租约存储（Redis 或内存）
    storage: Box<dyn LeaseStorage>,
    /// 当前租约
    current_lease: Arc<RwLock<Option<Lease>>>,
    /// 配置
    config: LeaseConfig,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Lease {
    /// 租约 ID
    pub id: Uuid,
    /// 持有者（智慧主控）
    pub holder: String,
    /// 创建时间
    pub created_at: DateTime<Utc>,
    /// 过期时间
    pub expires_at: DateTime<Utc>,
    /// 最后续约时间
    pub last_renewed: DateTime<Utc>,
}

impl LeaseManager {
    /// 检查租约有效性
    pub fn is_valid(&self) -> bool {
        let lease = self.current_lease.read().unwrap();
        match lease.as_ref() {
            Some(lease) => Utc::now() < lease.expires_at,
            None => false,
        }
    }
    
    /// 续约（智慧主控调用）
    pub async fn renew(&self) -> Result<()> {
        let mut lease = self.current_lease.write().unwrap();
        if let Some(ref mut lease) = *lease {
            lease.last_renewed = Utc::now();
            lease.expires_at = Utc::now() + self.config.lease_duration;
            self.storage.save(lease.clone()).await?;
        }
        Ok(())
    }
}
```

### 3. 心跳检测器（HeartbeatChecker）

**文件**: `src/watchdog/heartbeat.rs`

```rust
pub struct HeartbeatChecker {
    /// gRPC 客户端
    client: HeartbeatClient,
    /// 失败计数器
    failure_count: Arc<AtomicU32>,
    /// 配置
    config: HeartbeatConfig,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct HeartbeatStatus {
    /// 租约 ID
    pub lease_id: Uuid,
    /// 时间戳
    pub timestamp: DateTime<Utc>,
    /// 健康状态
    pub health: HealthStatus,
    /// 系统指标
    pub metrics: SystemMetrics,
    /// 最近错误
    pub recent_errors: Vec<String>,
}

impl HeartbeatChecker {
    /// 检查心跳
    pub async fn check(&self) -> Result<HeartbeatStatus> {
        match self.client.get_heartbeat().await {
            Ok(status) => {
                self.failure_count.store(0, Ordering::SeqCst);
                Ok(status)
            }
            Err(e) => {
                let failures = self.failure_count.fetch_add(1, Ordering::SeqCst);
                if failures >= self.config.max_failures {
                    Err(anyhow!("Heartbeat failed {} times", failures))
                } else {
                    Ok(HeartbeatStatus::degraded(format!("Check failed: {}", e)))
                }
            }
        }
    }
}
```

### 4. 诊断引擎（DiagnosticEngine）

**文件**: `src/diagnostic/analyzer.rs`

```rust
pub struct DiagnosticEngine {
    /// 日志分析器
    log_analyzer: LogAnalyzer,
    /// 模式匹配器
    pattern_matcher: PatternMatcher,
    /// 根因分析器
    root_cause_analyzer: RootCauseAnalyzer,
}

impl DiagnosticEngine {
    /// 分析故障，生成恢复计划
    pub async fn analyze(&self, status: &HeartbeatStatus) -> Result<RecoveryPlan> {
        // 1. 日志分析
        let log_patterns = self.log_analyzer.analyze(&status.recent_errors).await?;
        
        // 2. 模式匹配
        let known_issues = self.pattern_matcher.match_patterns(&log_patterns);
        
        // 3. 根因分析
        let root_cause = self.root_cause_analyzer.identify(&known_issues).await?;
        
        // 4. 生成恢复计划
        let recovery_level = self.determine_recovery_level(&root_cause);
        
        Ok(RecoveryPlan {
            level: recovery_level,
            root_cause,
            actions: self.generate_actions(&root_cause, recovery_level),
            retry_count: 0,
            backoff: Duration::from_secs(1),
        })
    }
    
    /// 确定恢复级别
    fn determine_recovery_level(&self, root_cause: &RootCause) -> RecoveryLevel {
        match root_cause.severity {
            Severity::Low => RecoveryLevel::L1_QuickFix,
            Severity::Medium => RecoveryLevel::L2_AIDiagnosis,
            Severity::High => RecoveryLevel::L3_HumanIntervention,
        }
    }
}
```

### 5. 恢复执行器（RecoveryExecutor）

**文件**: `src/recovery/executor.rs`

```rust
pub struct RecoveryExecutor {
    /// L1 快速修复
    quick_fix: QuickFixExecutor,
    /// L2 AI 诊断
    ai_diagnosis: AIDiagnosisExecutor,
    /// L3 人工介入
    human_intervention: HumanInterventionExecutor,
    /// 审计日志
    audit_log: AuditLogger,
}

impl RecoveryExecutor {
    /// 执行恢复计划
    pub async fn execute(&self, plan: RecoveryPlan) -> Result<()> {
        self.audit_log.log_recovery_start(&plan).await;
        
        let result = match plan.level {
            RecoveryLevel::L1_QuickFix => {
                self.quick_fix.execute(&plan.actions).await
            }
            RecoveryLevel::L2_AIDiagnosis => {
                self.ai_diagnosis.execute(&plan.actions).await
            }
            RecoveryLevel::L3_HumanIntervention => {
                self.human_intervention.execute(&plan.actions).await
            }
        };
        
        self.audit_log.log_recovery_result(&plan, &result).await;
        
        result
    }
}
```

### 6. 智慧主控心跳上报器

**文件**: `src/core/heartbeat_reporter.rs`

```rust
pub struct HeartbeatReporter {
    /// gRPC 客户端（连接核心主控）
    client: WatchdogClient,
    /// 租约 ID
    lease_id: Uuid,
    /// 上报间隔
    interval: Duration,
    /// 自检模块
    self_checker: SelfChecker,
}

impl HeartbeatReporter {
    /// 启动心跳上报
    pub async fn start(&self) {
        let mut interval = tokio::time::interval(self.interval);
        
        loop {
            interval.tick().await;
            
            // 1. 执行自检
            let self_check = self.self_checker.check().await;
            
            // 2. 构造心跳消息
            let heartbeat = HeartbeatMessage {
                lease_id: self.lease_id,
                timestamp: Utc::now(),
                health: self_check.health,
                metrics: self_check.metrics,
                errors: self_check.recent_errors,
            };
            
            // 3. 发送到核心主控
            if let Err(e) = self.client.send_heartbeat(&heartbeat).await {
                tracing::error!("Heartbeat report failed: {}", e);
                // 进入降级模式
                self.enter_degraded_mode().await;
            }
        }
    }
}
```

## 🔌 gRPC 协议定义

**文件**: `src/proto/watchdog.proto`

```protobuf
syntax = "proto3";

package newclaw.watchdog.v1;

// 心跳服务
service HeartbeatService {
    // 智慧主控上报心跳
    rpc ReportHeartbeat(HeartbeatRequest) returns (HeartbeatResponse);
    // 核心主控查询心跳状态
    rpc GetHeartbeatStatus(GetHeartbeatRequest) returns (HeartbeatStatus);
}

// 租约服务
service LeaseService {
    // 申请租约
    rpc AcquireLease(AcquireLeaseRequest) returns (Lease);
    // 续约
    rpc RenewLease(RenewLeaseRequest) returns (Lease);
    // 释放租约
    rpc ReleaseLease(ReleaseLeaseRequest) returns (ReleaseLeaseResponse);
}

// 恢复服务
service RecoveryService {
    // 触发恢复
    rpc TriggerRecovery(TriggerRecoveryRequest) returns (RecoveryResponse);
    // 查询恢复状态
    rpc GetRecoveryStatus(GetRecoveryRequest) returns (RecoveryStatus);
}

// 消息定义
message HeartbeatRequest {
    string lease_id = 1;
    int64 timestamp = 2;
    HealthStatus health = 3;
    SystemMetrics metrics = 4;
    repeated string recent_errors = 5;
}

message HeartbeatResponse {
    bool acknowledged = 1;
    int64 server_time = 2;
}

message HealthStatus {
    enum Status {
        UNKNOWN = 0;
        HEALTHY = 1;
        DEGRADED = 2;
        UNHEALTHY = 3;
    }
    Status status = 1;
    string message = 2;
}

message SystemMetrics {
    uint64 memory_mb = 1;
    float cpu_percent = 2;
    uint64 active_sessions = 3;
    uint64 request_rate = 4;
}

message Lease {
    string id = 1;
    string holder = 2;
    int64 created_at = 3;
    int64 expires_at = 4;
    int64 last_renewed = 5;
}
```

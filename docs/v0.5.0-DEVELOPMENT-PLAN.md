**1.2 嵌入客户端实现** (3 天)
- [ ] 创建 `src/embedding/mod.rs`
- [ ] 实现 `EmbeddingClient` trait
  ```rust
  #[async_trait]
  pub trait EmbeddingClient: Send + Sync {
      async fn embed(&self, texts: Vec<String>) -> Result<Vec<Vec<f32>>>;
      async fn embed_single(&self, text: &str) -> Result<Vec<f32>> {
          let mut results = self.embed(vec![text.to_string()]).await?;
          Ok(results.pop().unwrap())
      }
  }
  ```
- [ ] 实现 OpenAI 嵌入客户端

- [ ] 单元测试

**1.4 性能优化** (2 天)

```
│   ├── strategy.rs            # 策略引擎 (已有)
│   └── retrieval.rs           # 🆕 语义检索
│
├── embedding/                  # 🆕 向量嵌入模块
│   ├── mod.rs
│   ├── client.rs              # 嵌入客户端
│   ├── pipeline.rs            # 向量化流水线
│   └── cache.rs               # 嵌入缓存
│
├── core/
│   └── context.rs             # 上下文管理器 (已有，需重构)
│
└── gateway/
    └── context_integration.rs # 🆕 Gateway 集成
```

### 核心接口

```rust
// 统一的上下文管理器
pub struct ContextManager {
    token_counter: Arc<TokenCounter>,
    embedding_client: Arc<dyn EmbeddingClient>,
    vector_store: Arc<VectorStore>,
    summarizer: Option<Arc<dyn Summarizer>>,
    config: ContextConfig,
}

impl ContextManager {
    // 添加消息 (自动分块和嵌入)
    pub async fn add_message(&self, message: &str, source: &str) -> Result<String>;

    // 语义检索
    pub async fn retrieve(&self, query: &str, config: RetrievalConfig) -> Result<Vec<ContextChunk>>;

    // 构建上下文 (检索 + 截断 + 压缩)
    pub async fn build_context(&self, messages: Vec<Message>, max_tokens: usize) -> Result<Vec<Message>>;

    // 统计信息
    pub fn stats(&self) -> ContextStats;
}
```

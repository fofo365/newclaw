    config: BatchConfig,
    request_queue: mpsc::UnboundedSender<BatchRequest>,

    pub fn new(config: BatchConfig) -> Self;
    pub async fn submit(&self, text: String, estimated_tokens: usize)
        -> Result<EmbeddingResult, EmbeddingError>;
}

### 2. 缓存集成到 Pipeline
**文件**: `src/embedding/pipeline.rs` (更新)

**改动**:
- 添加 `cache: Option<Arc<EmbeddingCache>>` 字段
- 实现 `with_cache()` 方法
- 更新 `process_document()` 方法支持缓存
- 添加缓存统计：`cache_hits`, `cache_hit_rate`

**缓存策略**:
1. 对每个文本块计算哈希键
2. 检查缓存命中
3. 只嵌入未缓存的块
4. 将新结果存入缓存
5. 保持原始顺序合并结果

**性能提升**:
- 缓存命中时，延迟从 ~500ms 降至 < 1ms
- 目标命中率：> 80%

**状态**: ✅ 代码完成，编译中

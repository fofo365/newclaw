pub trait LLMProviderV3: Send + Sync {
    fn name(&self) -> &str;
    async fn chat(&self, req: ChatRequest) -> Result<ChatResponse, LLMError>;
    async fn chat_stream(&self, req: ChatRequest) -> Result<Stream<String>, LLMError>;
    fn count_tokens(&self, text: &str) -> usize;
    async fn validate(&self) -> Result<bool, LLMError>;
}

### 2. 多种模型切换策略
- **Static**: 固定模型
- **RoundRobin**: 轮询负载均衡
- **Fallback**: 故障降级
- **CostOptimized**: 成本优化（简单任务用便宜模型）
- **Adaptive**: 基于任务复杂度自适应

- 导致代码混乱，难以维护

**解决方案**:
1. **重构 `src/context/mod.rs`**
   - 移除重复的 ContextManager 定义
   - 保留 token_counter、truncation、strategy 模块
   - 重新导出 `core::context::ContextManager`

2. **增强 `src/core/context.rs`**
   - 集成 Token 计数功能
   - 集成截断策略引擎
   - 添加新的方法：
     - `count_tokens()` - 文本 token 计数
     - `count_messages_tokens()` - 消息列表 token 计数
     - `estimate_token_usage()` - token 使用估算
     - `apply_truncation_strategy()` - 策略化截断
     - `get_stats()` - 统计信息

3. **修复编译错误**
   - 添加 Debug trait to TokenCounter
   - 添加 Debug trait to TruncationStrategy
   - 添加 Debug trait to StrategyEngine
   - 添加 Hash/Eq traits to TruncationStrategyType

### 2. EmbeddingPipeline 缓存集成
**文件**: `src/embedding/pipeline.rs` (100+ 行新增)

**核心改进**:
- ✅ 添加 `cache: Option<Arc<EmbeddingCache>>` 字段
- ✅ 实现 `with_cache()` 构建器模式
- ✅ 更新 `process_document()` 支持缓存
- ✅ 缓存统计：`cache_hits`, `cache_hit_rate`

**缓存策略**:
```rust
// 1. 计算缓存键（哈希）
let cache_key = self.compute_cache_key(chunk);

// 2. 检查缓存
if let Some(result) = cache.get(&cache_key).await {
    return result;  // < 1ms
}

// 3. 嵌入并缓存
let result = client.embed_batch(chunk).await?;
cache.put(cache_key, result.clone()).await;
```
